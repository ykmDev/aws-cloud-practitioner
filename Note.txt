    1) Cloud Computing
      def: the practice of using a network of remote servers hosted on internet
        to store, manage, process data rather than local/personal server, computer.
          On-Premise:
            You own the servers, hire IT people, pay or rent the real-estate, take all risk.
          Clud Providers:
            someone else own servers, hire IT people, pay or rent real-estate,
            you are responsible for your configuration cloud services and code.
      evaluation:
        dedicated server:
          one physical machine dedicated to single a business, run single web-app.
          Note: very expensive, high maintenane, high Security.
        virtual private server(VPS):
          one physical machine dedicated to a single business, 
          one physical machine is virtualized into a sub-machines
          runs Multiple web-apps.
          Note: better utilization and isolation of resources.
        shared hosting:
          one physical machine shared by hundred of businesses,
          Note: very cheap, limited functionality, poor isonlation.
        Cloud Hosting:
          Multiple physical machines that act as one system.
          abstracted into multiple cloud services.
          Note: flexible, scalable, cost-effective, high configurablility.

        Cloud Service Provider(CSP);
          provite multiple services.
            those can be chained together to create cloud Architecuture.
            those are accessable via single unified api eg: aws api.
            those utilized metered billing based on usage eg: per second, per hour.
            those have rich monitoring eg: aws cloudtrail.
            those have IaaS offering.
            those offers automation via IaC.
          Note:
            if a company offers a multiple cloud services under a single UI 
            but do not meet most of all these requirements, it would be referred to as Cloud Platform eg: twilio, HashiCorp, Databricks.
